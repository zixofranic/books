# Hey — thanks for giving this a listen

This is the audiobook of *Just Predicting Words*. Really appreciate you
taking the time.

The narrator you're hearing is a custom TTS agent I built specifically
to read this book — honestly one of the most fascinating things I've
ever put together.

The file is a single `.m4b` (~254 MB) with chapters and cover art. Skip
Apple Books — sideloading on it is a headache. Use one of these instead:

### iPhone / iPad
1. Install **BookPlayer** from the App Store (free).
2. Open the link I sent in **Safari** → tap **Download**.
3. Files app → **Downloads** → tap the file → share → **BookPlayer**.
4. Open BookPlayer and hit play.

### Android
1. Install **Smart AudioBook Player** (free) from the Play Store.
2. Open the link in Chrome → tap **Download**.
3. Move the file from `/Download` into `/Audiobooks`.
4. Open the app — the book shows up. Hit play.

### Mac / PC
VLC plays it. On Mac, double-click also opens it in Books.
